## ğŸš€ Features

- **Sidebar Navigation**
  - Quick access to `Loads`, `Statements`, and `Transactions`.
  - Simple logout button for session management.

- **Company & Account Selection**
  - Dropdowns for choosing different companies (e.g., Test Yantra, Tech Pyramyd).
  - Account dropdown to view associated accounts.

  - Displays the current account balance dynamically.
  - Visual display with relevant icons or images.

- **Loads Table**
  - Shows recent transaction loads including:
    - Date
    - Credit amount
    - Account balance
    - UTR/RRN number
    - Account number / UPI ID
  - Data is dynamically populated via JavaScript (`j.js`).

  - Mobile-friendly footer with **Sign Up** and **Login** buttons.

## âš™ï¸ How It Works

1. **User Interface**
   - When the page loads, users are presented with a sidebar and a dashboard area.
   - Dropdowns allow selecting a company and its corresponding account.

2. **Dynamic Updates**
   - When a company/account is selected, JavaScript (`j.js`) updates the displayed balance and loads data into the table body (`#loadsTableBody`).

3. **Responsive Design**
   - Optimized for both desktop and mobile views.
   - Footer buttons (`Sign up`, `Log in`) appear for mobile screens.

4. **Future Enhancements**
   - Integrate real backend API for fetching live data.
   - Add authentication for login and logout.
   - Include charts for visual insights (balance trends, load statistics).
